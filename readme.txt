This is project for academic purpose only.

RestServer is the simple rest application which products some rest data on certain API's.

REST data are available in both XML and JSON format. 

Rest Client is another application which consumes the API from RestServer application using proxy pattern and populates data in web.

RestClient application also defines the Client Server Service Pattern. It defines how to make a legacy application to SOA frindly application doing the least changes on it.
